Settings: Unit & Item Amounts

Customize how much fluid one ingot or mold requires, plus how much molten metal each item contributes when melted. These values are used by every calculator, planner, solver, and capacity warning.

Label shown in app Fluid amount (mB) Use in quick selectors

Examples: if one ingot mold needs 100 mB, set “Ingot / Mold” to 100. If a melted item gives 10 mB, add it as a custom unit so rows can be entered directly in that item count.
